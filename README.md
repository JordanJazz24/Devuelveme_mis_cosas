# Devuélveme mis cosas
Gestor inteligente de préstamos personales — EIF411, Universidad Nacional de Costa Rica

## Funcionalidades implementadas
- Registro de préstamos con foto, contacto de agenda y categoría
- Indicadores de urgencia por color (verde/amarillo/rojo)
- Recordatorios vía WhatsApp o SMS con mensaje personalizable
- Notificaciones locales automáticas (7 días antes y al vencer)
- Historial de préstamos devueltos con foto de devolución
- Exportar e importar datos en JSON para respaldo
- Modo oscuro/claro manual

## Stack tecnológico
Kotlin · Jetpack Compose · MVVM · Room · Hilt · WorkManager · Navigation Compose · Coil · Material 3

## Requisitos
Android 8.0+ (API 26)
